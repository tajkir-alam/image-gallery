
# Project Title

This front-end website Called Image Gallery. 


## Live Link

https://image-gallery-project-dnd.netlify.app/

## Technology & Packages:
    1. React.Js
    2. React-Icons
    3. Tailwind
    4. DaisyUi


## Run Locally

Clone the project

```bash
  git clone https://github.com/tajkir-alam/image-gallery.git
```

Go to the project directory

```bash
  cd image-gallery
```

Install dependencies

```bash
  npm install / npm i 
```

Start the server

```bash
  npm run dev
```
